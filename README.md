## Кейс "Эффективность"
**Бот "Помощник для запоминания теории по кривой Эббингауза"**
Пользователю доступна возможность загрузки собственных вопросов с ответами.
Бот предлагает пройти викторину с временным промежутком, который определяется кривой Эббингауза.
Бот поможет пользователю закрепить теорию и потренировать свою память.
